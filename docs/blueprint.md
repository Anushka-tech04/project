# **App Name**: IP Threat Defender

## Core Features:

- IP Address Lookup: Allow users to enter an IP address and retrieve location information and other associated details.
- Threat Information: Integrate with open-source threat intelligence APIs to provide risk scores and threat classifications for IP addresses. The LLM acts as a tool to aggregate risk assessments and incorporate relevant data from threat intelligence feeds into its analysis of risk.
- Malware Hash Lookup: Enable users to submit malware hashes (MD5, SHA1, SHA256) and query open-source databases to identify known malware.
- Report Generation: Generate reports summarizing the IP address information, threat assessment, and malware analysis.

## Style Guidelines:

- Primary color: Deep Blue (#2962FF) to convey security and reliability.
- Background color: Light Gray (#F0F4F8) to provide a clean, non-distracting backdrop.
- Accent color: Bright Orange (#FF6B6B) to highlight critical threat information and warnings.
- Body and headline font: 'Inter' sans-serif font for clear readability and a modern aesthetic.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist, geometric icons to represent different types of threat information.
- Subtle animations to draw attention to changes in threat levels or new findings.
